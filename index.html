🤖 Cấu hình Telegram ▶

📢 Chat ID chung (dùng cho tất cả thông báo)
Chat ID nhóm/channel dùng chung cho toàn bộ bot token bên dưới. Lấy Chat ID tại đây
📢 Thông báo chung (đơn hàng, chi phí, ...)
Token từ @BotFather trên Telegram. Dùng chung Chat ID bên trên.
🔒 Thông báo chốt ca
Token riêng cho bot gửi báo cáo chốt ca. Dùng chung Chat ID bên trên. Tạo bot tại @BotFather
⚠️ Cảnh báo (tồn kho, xóa bàn, xóa món)
Token riêng cho bot gửi cảnh báo tồn kho, xóa bàn, xóa món. Dùng chung Chat ID bên trên. Tạo bot tại @BotFather
📊 Chi phí
Token riêng cho bot gửi thông báo khi có chi phí phát sinh. Dùng chung Chat ID bên trên. Tạo bot tại @BotFather
